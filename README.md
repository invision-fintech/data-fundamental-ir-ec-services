# data-fundamental-ir-ec-services

A shared space for applied data engineering / data analysis work — proofs of
concept, small research artifacts, self-contained tasks. It's a mix of things,
and it'll keep growing.

## Structure

Each task lives in its own folder under `tasks/`, and is self-contained: its
`README.md` has everything you need to understand and complete it.

```
tasks/
  <task-slug>/
    README.md       what the task is and what "done" looks like
    ...             starter code, notebooks, whatever the task needs
```
